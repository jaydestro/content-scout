Content Scout

Setup

1. Pick an agent

Content Scout runs as prompts against an AI coding agent. Pick which one the web UI should invoke when you press Start run. You can change this any time.

Use {prompt} as the placeholder for the Scout command.

Note: SCOUT_RUNNER env var is set — it overrides any choice saved here. Unset it and restart to edit.

2. Create a product config

Content Scout needs a config file describing what to search for. Run the onboarding wizard one of these ways:

  • In your editor: open Copilot Chat / Claude Code / Cursor in this repo and say /scout-onboard (or "scout onboard")
  • Standalone CLI: 
    cd tools/onboard-cli
npm install
node bin/onboard.js
  • From this UI (needs a configured agent above):

3. Add API keys (optional but recommended)

Some sources (YouTube, Reddit, Bluesky, X) need API keys. Values are saved to .env at the repo root. Leave any you don't have blank — those sources are skipped during a scan.
