      Networks & output
      
              
      
                Networks to scan  · 
                
      
                  
                  
                  
                  
                  
                  
                  
                  
                  
                
      
                
      
                  How X / LinkedIn / Reddit get scanned: Content Scout tries the best method available, then falls back if it can't.
                  The best method is the sign-in scan — it quietly reads results from a browser you've already signed in to, so you get the same posts you'd see while logged in (replies, ranked feed, members-only subreddits). It never touches passwords or DMs.
                  If that's not set up, it falls back to public search (via your Brave Search key) and then to RSS feeds.
                  Just check the network here — you don't pick the method. To turn on the sign-in scan, open the
                  Run view and click “Open browser & sign in” once.
