                Networks to scan  · 
                
      
                  
                  
                  
                  
                  
                  
                  
                  
                  
                
      
                
      
                  How X / LinkedIn / Reddit are scanned: the agent uses a layered cascade per platform.
                  Layer 0 (recommended) = logged-in browser scan via the
                  tools/browser-scan/ tool, opt-in. Falls back through Brave Search, RSSHub, and the public-API
                  layers when no fresh sidecar exists. You don't have to choose a method here — just check the network and
                  the agent will use whatever Layer it can. To enable Layer 0, open the
                  Run view and click "Open browser & sign in" at the top once.
