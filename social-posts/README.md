# Social posts

Content Scout writes generated social posts and posting calendars to this directory.

- `{YYYY-MM-DD-HHmm}-{slug}-social-posts.md` — bulk posts from a scan report
- `{YYYY-MM-DD-HHmm}-{slug}-solo-{url-slug}.md` — one-off post from a single URL
- `{YYYY-MM-DD-HHmm}-{slug}-posting-calendar.md` — weekly schedule
- `images/` — generated thumbnails and brand assets

All generated content here is gitignored. Only this README and the `images/brand/` whitelist are tracked.
