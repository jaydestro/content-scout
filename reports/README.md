# Reports

Content Scout writes scan reports and trend reports to this directory.

- `{YYYY-MM-DD-HHmm}-{slug}-content.md` — scan reports
- `{YYYY-MM-DD-HHmm}-{slug}-content.json` — machine-readable sidecar
- `{YYYY-MM-DD-HHmm}-{slug}-trends.md` — trend reports
- `.seen-links.json` — dedup tracker
- `.scout-state/` — internal run state

All generated content here is gitignored. Only this README is tracked.
